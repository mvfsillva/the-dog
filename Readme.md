## Getting Started

### How to Run

```sh
$ npm install  # install dependencies
# after
$ npm start # Run application
```
> Open browser to http://localhost:3000


### Running the tests

```sh
$ npm run test
$ npm run test:watch
```

### Coding style tests

```sh
$ npm run lint
$ npm run lint:css
$ npm run lint:all
```

### Coding style format

```sh
$ npm run format
$ npm run format:check
```
